Minecraft BE アドオン/リソースパック一覧

マイクラBE で使用可能な Yuka0745作 のアドオンなどが配布されています!

Gote-Supporter
マイクラを快適に便利に楽しめるように設計されたUIリソースパックです!
Hack Item Giver
Hackなどを使わずに通常では出せないアイテムやブロックを出せるようにするアドオンです!
Custom UI Maker
本と羽根ペンを使って内容を自由に入力して表示できるUIを作ることができます!
Legacy GameConsoleUI
コンソールエディション(WiiU, 旧Switch, PS3, 旧PS4, Xbox360, 旧Xbox One)のようなUIにするリソースパックです!
Chat Text Copy
チャットの文章を簡単にコピペできます
Preview デバッグテキスト切り替え
Preview版の上のデバッグテキストを非表示にしたりできます。
HUD画面でF10を押して切り替えます。
タッチ操作でかんたんF5
F5キーをスマホで押せる！
camera コマンドの実験を有効化する必要があります。
Custom boss bar UI
カスタムボスバーをscoreboardのsidebarで作れます。
/scoreboard objectives add <ID> dummy
Fullscreen Bock Screen
Gote-Supporterにあったフルスクリーンの本画面です。
Fullscreen Play Screen
Gote-Supporterにあったフルスクリーンの遊ぶ画面です。
SNS Feed Screen
Gote-SupporterにあったSNSみたいなフィード画面です。
グリッド表示のサーバーフォーム
サーバーフォームをグリッド表示にします
Newline Text for Anvil
金床で改行テキストを使用できます。
チャットでセクション装飾消す
チャットでのセクション装飾を消します
全てのサーバーで動くわけではありません
World Icon Replacer
ワールドのアイコンを置き換える
Tab Key Playerlist UI
このマイクラ統合版用リソースパックは、
HUDでTabキーを押すと、プレイヤーリストを表示することができます。

ほかにもいろんなアドオンをGithubのリポジトリに配布しています！
