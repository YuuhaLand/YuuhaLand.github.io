遊び方
ラウンドが始まると、15秒間の無敵時間が始まります。
無敵時間中は、ダメージを受けることはありません。

無敵時間が終了すると、バトル開始です。
30秒ごとにチェストの中身が更新されます。
チェストの道具をうまく使い、生き残りを目指しましょう！
※「すべて取る」ことはできません。

残り2分30秒になると、「ショーダウン」が開始します。
ショーダウンでは、プレイヤーのネームタグが見えるようになります。

最後の1人になったプレイヤーが勝利となり、勝利数カウントにカウントされます。
制限時間を切る前に決着がつかなかった場合ランダムなプレイヤーが勝利となりますが、勝利数カウントにはカウントされません。

試合中に脱退した場合、負けとしてみなされ、すぐに入りなおしてもバトルに途中参加することはできませんのでご注意ください。

倒されると、観戦モードになります。
マップを自由に飛びバトルを観戦することができます。
フィールドから出ることはできません。

また、放置モードはありません。全員参加となるのでご注意ください。
